Analyze the provided document image to determine if it is of sufficient quality

### CRITICAL CHECKLIST
1. **LEGIBILITY**: Is the text sharp and readable? Is it too blurry, pixelated, or dark to extract numbers accurately?
2. **COMPLETENESS**: Is the document cut off? Are the edges of the tables visible? (e.g., Is the 'Amount' column missing?)
3. **RELEVANCY**: Is this actually a financial document? (Flag images of IDs, cats, landscapes, or unrelated personal photos).
4. **ORIENTATION**: Is the document sideways or upside down? (Note: We can rotate it, but flag it if it affects readability).
5. **TAMPERING**: Does the document look digitally altered or "photoshopped" in a suspicious way?
6. **RECENCY**: Briefly check if the dates visible are within the last 2-3 years. If the document is from 2015, flag it as 'rejected' for being outdated.

### EVALUATION CRITERIA
- **is_ready_for_analysis**: Set to TRUE only if the document is clear, complete, and relevant.
